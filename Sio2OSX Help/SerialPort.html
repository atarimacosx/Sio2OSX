Near the bottom center of the Sio2OSX program window is the serial port status.  It will show which serial port is currently in use, and allow the user to chose an alternate one from the dropdown menu, if more than one USB serial adapter is installed.



If no serial ports are found on the system, the dropdown for the SIO Port will display "None".  Note that if you start Sio2OSX without the USB Serial adapter plugged in, you may plug it in after startup, and the program will automatically recognize it and open that serial port.  However, it is advised to not unplug the USB serial adpater while it is the active serial port for Sio2OSX, as this may cause your Macintosh to lock for 10-15 seconds.  This is a function of the serial driver, and cannot be controlled by the Sio2OSX program.

The SIO Speed display shows the current baud rate being used to communicate between the Atari and the Macintosh.  This starts at 19200 for any disk data transfers.  In the General tab of Sio2OSX's Preferences, you are able to specify the maximum SIO baud rate, and this defaults to 57600, 3x.  This is used by DOS's such as SpartaDOS and modified MyDos (APE Warp) to speed up data transfers.  If you want to limit the maximum rate to 1x or 2x, you can do so by changing the preferences.  Cassette transfers take place at 600 baud, and the display will change to reflect this when you are playing a cassette tape image in Sio2OSX.
