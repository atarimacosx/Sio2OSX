This tab allows the user to specify the general preferences for Sio2OSX.  



Ignore Header Write Protect Bit On ATR Disk Images

 This checkbox should be checked to ignore the Header Write Protect Bit that is set in some ATR disk images.  It is unchecked by default, and should only be checked if the user needs to be able to write to an ATR image that has the bit set. 

Maximum SIO Speed

 This dropdown allows you to set the maximum speed that the SIO/serial interface between the Mac and the Atari runs at.  The normal speed of an Atari SIO interface is 19200.  Modified OS systems such as Sparados, and APE Warp MYDOS are able to run at a higher speed.  If you would like to use these high speed features, set the maximum speed to 57600 (3x) or 38400 (2x).  If you have issues running at these higher speeds, then lower the speed back to 19200. 


SIO Hardware Type

 This dropdown selects which RS232 data line is used by the SIO interface.  Most SIO interfaces use RI so this is the default.  If your SIO interface hardware uses DSR or RTS, use this to select those lines. 


Delay Factor

 This dropdown sets a delay that is inserted between parts of the SIO protocol.  This should be left at "No Delays" for a USB serial interface.  This would only be useful if you were using a PCI serial board, which at this time the author does not know of any working interfaces of this type on OSX.
